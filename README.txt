# Всем привет, это мой проект по fastapi + asinco + docker. 

# первым делом нужно создать своё виртуальное окружение. Делается это команадной в  терминале. 
python -m venv ./venv/.
#После выполнения данного скрипта нужно зайти в виртуальное окружение. В терминале оно будет выделяться, 
# перед директорией будут стоять зеленые скобки и в нем текст. 

# после этого мы должны устанвить зависимости, делаетяся это при помощи команды ниже в терминале: 
pip install -r requirements.txt

#После, для того если мы хтим запустить наш проект на своём компьютере, то нам нужно ввести такую команду в терминале: 
uvicorn.app.mainn:app
# тогда сервер будет запущен 

# Eсли мы хотим запустить через докер (мы должны находиться в директории нашего проекта), то ниже команда:  
docker-compose up 
# После этого проект соберется и контейнер автоматически активируется 
# чтобы выключить проект, то просто нажмите в терминале ctr+c
